# Unity Optimizer Pro

Unity Optimizer Pro is a simple and powerful Unity editor tool that improves your game's performance with minimal effort.

## Features

- One-click performance boost
- Smart light optimization
- GPU instancing automation
- Performance scoring and FPS estimation
- Platform presets (PC, Mobile, VR)
- Visual heatmap debugging

## How to Use

1. Open the tool in Unity
2. Click "Scan Scene"
3. Click "Boost Performance"
4. Compare results
